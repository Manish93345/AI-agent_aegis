# AEGIS - Adaptive Embedded Guardian & Intelligent System

## Overview
A personal AI assistant with voice control, automation, and security features for your laptop.

## Features
- Voice-controlled automation
- System monitoring and learning
- Multi-level security system
- User authentication
- Routine automation

## Installation
1. Clone repository
2. Run: `pip install -r requirements.txt`
3. Configure settings in `config/` folder
4. Run: `python main.py`

## Quick Start
```bash
# Activate virtual environment
aegis_venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run AEGIS
python main.py
